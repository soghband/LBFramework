What's LightBreak Framework can do?
  • Flexible to using controller or html.
  • Easy config route and filter data.
  • Auto optimization image by GD
  • Auto caching when switch to production mode.
  • Session process while caching if you need.
  • Time controller tool make you know how lag in which process.
  • Dev IO real-time build CSS like SASS.

  • Controller only when system not found html file in view folder Example

Easy config route and filter data
Using JSON for config route and filter.
Example1 Filter url pattern 
{
    "filterUrl/example": {
        "controller":"filterUrlExample1"
    },
    "filterUrl/{ex1}/{ex2}": {
        "controller":"filterUrlExample2",
        "url_filter":{
            "{ex1}": "^[a-z]{3}$",
            "{ex2}": "^[A-Z]{3}$"
        }
    }
}

Dev IO real-time build CSS like SASS.
Dev IO is NodeJS and Socket.io help for real-time update css while you coding stylesheet. For this framework there can't use SASS because system need to combine CSS and JS, So i build this module for real-time update stylesheet instead.
How to use Dev IO
1. Change config "ENV_MODE" to "dev" and "ENABLE_DEV_IO" to "true".
2. Make sure jou installed NodeJS and go to folder "dev".
3. Run "npm install".
4. Start Dev IO by command "node start" and refresh page.
Then enjoin css coding.
