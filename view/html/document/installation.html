Installation
Server Requirements
  1. Apache >= 2.2
  2. PHP >= 5.6.0
  3. OpenSSL PHP Extension
Download Sourcecode
Framework sourcecode is free in GitHub you can clone by git or download to your computer.
Root Document
Root document can set to framework public or root of framework folder. Use .htaccess to managing url.
Composer
Make sure you have composer in your computer and go to project root then update it.
[PROJECT_ROOT]> composer update
Other way: download composer.phar into framework folder and run command
[PROJECT_ROOT]> php composer.phar update
