Configuration
ENV_MODE
dev: Development mode disable caching and not write resource file (image, JS, CSS)
any: Enable caching and write resource file (image, JS, CSS)
TIME_CHECK
true: Enable time check work with Environment Development mode
false: Disable time check;
JS_PATH
any: Javascript path location
JS_COMPRESS
true: Enable Minify and Combine Javascript
false: Disable Minify and Combine Javascript
CSS_PATH
any: Style Sheet path location
CSS_COMPRESS
true: Enable Minify and Combine Style Sheet
false: Disable Minify and Combine Style Sheet
DEFAULT_TITLE
any: Set default for website's title.
RAW_IMAGE_PATH
any: Image path location
ENABLE_DEV_IO
true: Enable DEV_IO working with Environment Development mode
false: Disable DEV_IO
JPG_QUALITY
0 to 100: Quality of JPG image optimization.
PNG_COMPRESS_LEVEL
0 to 9: PNG Compression level
SESSION_SECURE
true: Enable Session secure
false: Disable Session secure
